
```

```

Installing necessary packages and libraries:

```
$ yarn install
```

Running a development server locally:

```
$ yarn start
```

Navigate to either your [Local Port](http://localhost:3000) or [Network Port](http://10.0.0.7:3000) to see the application running.
